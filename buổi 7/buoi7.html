CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ dùng để mô tả cách trình bày của tài liệu HTML. CSS giúp tách biệt nội dung (HTML) và cách trình bày (CSS), cho phép bạn kiểm soát giao diện của trang web một cách linh hoạt và hiệu quả.
